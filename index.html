Metode LVQ

Learning Vector Quantization (LVQ) adalah algoritma klasifikasi berbasis jaringan saraf tiruan yang mengelompokkan data berdasarkan kemiripan dengan data latih. Input jawaban kuisioner (0–3) dinormalisasi, kemudian dihitung jarak Euclidean ke vektor prototipe untuk setiap kelas penyakit. Penyakit dengan jarak terdekat mendapat confidence score tertinggi.
